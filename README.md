# Мобильное приложение для дистанционного обучения

## Требования и принцип работы:
- Приложение выполнено с помощью фреймворка Flutter на языке Dart, работает на базе API (DRF), для использования необходимо Android-устройство, Интернет-соединение и ~100 МБ свободного места.
- Для доступа ко всему заявленному функционалу необходимо создать учетную запись и указать при регистрации почту, к которой имеется постоянный доступ.
- Данные синхронизированы с аккаунтом слушателя курса на веб-сайте.

## Ролевая модель доступа:
- Приложение предоставляет две роли - неавторизованный пользователь и авторизованный (роль пользователя - слушатель курсов).
- Неавторизованный пользователь имеет доступ к форме регистрации, восстановления пароля и авторизации в систему.
- Авторизованный пользователь с ролью слушатель курсов имеет доступ ко всему функционалу мобильного приложения.

## Доступные экраны:
- авторизация,
- регистрация,
- восстановление пароля,
- главная,
- каталог курсов,
- детальная информация о курсе,
- материалы курса (лекции, тесты, практические задания),
- профиль пользователя,
- настройки (с возможностью смены темы, шрифта, пароля и данных профиля),
- список курсов пользователя (с отображением прогресса по курсам),
- список полученных сертификатов,
- детальная информация о сертификате,
- сдача практического задания,
- прохождение теста,
- результаты тестирования с отображением выбранных вариантов и их правильности,
- общая статистика пользователя по курсам (с прогрессом обучения),
- посты-объявления курса с возможностью оставлять комментарии,
- избранные курсы,
- деактивация аккаунта;

## Функциональные возмжности авторизованного пользователя:
- просмотр каталога курсов,
- просмотр списка популярных курсов,
- просмотр статистики платформы,
- поиск, фильтрация и сортировка по каталогу курсов,
- запись на курс и оплата курсов,
- смена темы приложения (темная и светлая),
- смена размера шрифта основных элементов (мелкий, стандартный, крупный),
- редактирование профиля (ФИО, почта, имя пользователя),
- просмотр детальной информации по курсу,
- добавление и удаление курса из избранного,
- просмотр избранных курсов,
- просмотр своего профиля (с историем оплат курсов, чеками за каждую и общей потраченной суммой)
- выход из аккаунта,
- при наличии 50% прогресса по выбранному курсу - добавление отзыва на курс с количеством звёзд от 1 до 5,
- удаление и редактирование собственного оставленного отзыва,
- смена пароля,
- оплата курса,
- запись на бесплатный курс,
- просмотр материалов курса (лекции, практические задания, тесты, посты),
- добавление комментария под пост-объявление,
- удаление комментария под постом-объявлением,
- открытие ссылки на видео-встречу курса (при наличии),
- сдача практической работы и просмотр детальной информации по ней,
- прохождение тесты (выбор вариантов ответа, просмотр результатов по тесту),
- просмотр всех пройденных тестов и выбранных вариантов ответов,
- просмотр списка курсов с указанием прогресса, на которые записан пользователь,
- просмотр общей статистики и прогресса по обучению,
- просмотр полученных сертификатов,
- скачивание материалов курса (лекции, практические задания, сертификат),
- просмотр детальной информации по полученному сертификату,
- деактивация аккаунта.
